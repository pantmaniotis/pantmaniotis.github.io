Welcome to my website

Hi, I am Pantelis Maniotis, and I have a passion for technology. This is my personal website where I host a number of data science projects I have worked on.

House Prices

In this project, we use several machine learning algorithms for regression (Ridge, Lasso, Elastic Net, XGBoost), in order to predict the sale price of residential homes in Ames, Iowa.

Movie recommender

In this project, I created a content based recommender system, using the Flash web app framework in Python. The posters of the movies were found using the OMDb API.

Titanic survivors

In this project, we use several machine learning algorithms for classification (Decision Tree, Random Forest, KNN, Naive Bayes, SVM), to estimate the probability of a titanic passenger to survive.

Digits Recognizer

In this project, we use a Neural Network to identify the digits from a tens of thousands of handwritten images, obtained from the MNIST dataset.

Edge caching (1) [pdf]

In this project, we examine the caching of VR applications in 5G networks, using combinatorial optimization techniques (Integer Linear Programming (ILP), Lagrange relaxation, etc).

Edge caching (2) [pdf]

In this project, we use machine learning techniques, i.e., Neural Networks, Deep Learning, to deliver VR applications in 5G networks, for a Video on Demand setting.

Edge caching (3) [pdf]

In this project, we use machine learning, i.e., LSTM networks for time-series forecasting, in order to optimize the delivery of VR applications in 5G networks for live streaming.
