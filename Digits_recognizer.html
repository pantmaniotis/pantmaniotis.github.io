1. Introduction

The aim of this project is to correctly identify the digits from a tens of thousands of handwritten images, obtained from the MNIST ("Modified National Institute of Standards and Technology") dataset. To achieve this goal, we will use a Neural Network. More information about this dataset can be found on Kaggle.

3. Employed Neural Network

In this Section, we initially transform our data, and then we feed the transformed data to our employed model.

In [8]:
from keras.utils.np_utils import to_categorical
y_train= to_categorical(y_train)
X_train = X_train.reshape(X_train.shape[0], -1)

Here, we normalize our data, in order to have values between 0 and 1.

In [9]:
#Normalize data
X_train = X_train / 255
X_test = X_test / 255
In [10]:
from keras.layers import Dense
from keras.models import Sequential
from keras import utils
from sklearn.model_selection import train_test_split
In [11]:
X_train, X_val, y_train, y_val = train_test_split(X_train, y_train, test_size=0.2, random_state=4)
In [12]:
model = Sequential()
model.add(Dense(512, activation='relu', input_shape=(784,)))
model.add(Dense(512, activation= 'relu'))
model.add(Dense(10, activation='softmax'))
model.compile(optimizer='adam', loss='categorical_crossentropy', metrics=['accuracy'])

Now, we train our employed Neural Network.

In [13]:
model.fit(X_train, y_train, validation_data=(X_val, y_val), epochs=5)

4. Results

In this section, we will use the employed Neural Network, in order to predict the hand written digits of the test set. Finally, we will present the achieved performance of our method.
