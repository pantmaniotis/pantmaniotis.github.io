1. Problem Description

The aim of this project is to predict the sale price of residential homes in Ames, Iowa. To this end, we use several regression algorithms, (i.e., Ridge, Lasso, Elastic Net, XGBoost), in order to achieve the lowest prediction error. The metric used in this project is the Root Mean Squared Error (RMSE). More information about the description of this project and the dataset used can be found on Kaggle.

4.2 Filling of missing values - Train set

Initially, we fill the missing values of the numerical attributes. To do so, we check the descriptive statistics of each attribute. By doing so, we can tell whether an attribute may be filled with zero (absence of this it) or not. When it cannot, e.g. GarageYrBlt, we use our intuition to fill the missing values by exploring relevant attributes.

In [20]:
#Numerical attributes
train['LotFrontage'] = train.groupby('Neighborhood')['LotFrontage'].transform(lambda x: x.fillna(x.mean()))
train['MasVnrArea'] = train['MasVnrArea'].fillna(0)
train['GarageYrBlt'] = train['GarageYrBlt'].fillna(train.YearBuilt)

Following a similar logic, we initially check which one of the categorical attributes can be filled with 'None'. By carefully observing the unique values of each attribute, we note that the attribute 'BsmtExposure' has one of it's values asigned as 'No'. For this reason, we declare the absence of this attribute with 'No' instead of 'None'.

In [21]:
#Categorical attributes
train['BsmtExposure'] = train['BsmtExposure'].fillna('No')
train['Electrical'] = train['Electrical'].fillna(train.Electrical.mode()[0])
train[miss_values_cat_train] = train[miss_values_cat_train].fillna('None')

x_test[skewed_feats] = np.log1p(x_test[skewed_feats])
